💡 実装ポイント

  • ・ 3ページ共通：上部に 14日分の日付ストリップ を追加（ダッシュボードと同じ）
  • ・ 選択日に応じて「その日の記録」を表示。合計値もその日のものに
  • ・ 一覧項目タップ → 下から出てくる ボトムシート編集モーダル（食事と同じUI）
  • ・ 食事：分量スライダー（10〜200%）→ PFC再計算
  • ・ 活動：時間スライダー（5〜180分）→ 消費カロリー再計算
  • ・ 体重：数値直接編集（kg と 体脂肪率）
  • ・ 3種全て削除ボタン付き（2段階確認）
  • ・ Today ボタンは「Today 4/11」のように実日付を併記（日付バグ検知用）
